# calcAttributes

provides attributes of different products

## Usage

``` r
calcAttributes(subtype = "Products")
```

## Arguments

- subtype:

  subtype of readProductAttributes function.

## Value

List of magpie objects with results on global level, empty weight, unit
and description.
